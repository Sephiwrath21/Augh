# Augh!

A delightfully pointless little website. Poke around — everything does something.

## 🌐 Live site

**https://sephiwrath21.github.io/Augh/**

Hosted on GitHub Pages (served from the `add-arcade` branch). Every push to that branch redeploys the site.

## Run it locally

Open `index.html` in any browser. No build step, no dependencies.

## Features

- 🎉 Confetti physics burst with a celebration fanfare
- 🔮 Random (slightly silly) quotes
- 🎨 Live color-palette reroll
- 🖱️ A click-counter with rank-ups
- 🧙 An interactive wizard that casts spells (with sound)
- 🔊 Sound-button corner
- 🕹️ [Augh Arcade](games.html) — 21 tiny games (Tetris, Uno, Battleship, Solitaire, Snake, 2048, and more)
- 🙌 [6 7](67.html) — the only emote that matters
- ✨ Full Y2K aesthetic across every page
