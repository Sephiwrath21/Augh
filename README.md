# Augh!

A delightfully pointless little website. Poke around — everything does something.

## Run it

Open `index.html` in any browser. No build step, no dependencies.

## Features

- 🎉 Confetti physics burst
- 🔮 Random (slightly silly) quotes
- 🎨 Live color-palette reroll
- 🖱️ A click-counter with rank-ups
